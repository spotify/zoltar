    Featran

    Featran dependency is not included by default. To use it you need to include an extra dependency.

    Maven
    <dependency>
  <groupId>com.spotify</groupId>
  <artifactId>zoltar-featran</artifactId>
  <version>0.3.1</version>
</dependency>
    Gradle
    dependencies {
  compile group: 'com.spotify', name: 'zoltar-featran', version: '0.3.1'
}
    sbt
    libraryDependencies += "com.spotify" % "zoltar-featran" % "0.3.1"
